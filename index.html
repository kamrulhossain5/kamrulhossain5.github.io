Project Link

PicEvent - Scan Event Flyers to Your Calendar

PicEvent is an app that quickly lets users add events to their iOS calendar using the camera or existing photos of event flyers without manually inputting information.

The app scans the event flyer picture (either from the gallery or the camera) for text, in which the text is then automatically inputted into its corresponding fields. The user is also able to manually enter or fix the information. After the user presses the "Add Event" button, the event is added to their phone's calendar.
